# Luxor-Semantic-Web-App

### Dependencies:

- Java
- Ruby ( to run the seed using `s-put` script, otherwise, you will need to manually seed or do it from the UI )

### Installing Fuseki:
Run `install.sh` to download and setup Fuseki server

### Running the server
Run `serve.sh`

### Seeding with our ontology
Run `seed.sh`
